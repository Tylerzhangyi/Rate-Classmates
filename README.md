# Rate My Classmate - 评价同学平台

一个基于 Vue 3 的学生评价平台前端应用。

## 功能特性

- ✅ 用户注册和登录（支持学号或邮箱）
- ✅ 学生列表浏览和搜索（按学校、年级、班级、姓名、平均分筛选）
- ✅ 评分系统（5个等级：夯、顶级、人上人、NPC、拉完了）
- ✅ 评论功能
- ✅ 排行榜（综合榜、学校榜）
- ✅ 个人中心（查看个人信息、统计、徽章、评分记录）

## 技术栈

- Vue 3 (Composition API)
- Vue Router 4
- Pinia (状态管理)
- Vite (构建工具)

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

## 使用说明

### 登录
- 使用测试账号登录：
  - 账号：`student001` 或 `student002`
  - 密码：`123456`

### 注册
- 填写账号、密码、姓名、学校、年级、班级信息即可注册
- 注册后会自动登录并添加到学生列表

### 评分
- 在学生列表或学生详情页点击"评分"按钮
- 选择评分等级（夯、顶级、人上人、NPC、拉完了）
- 可选择性添加评论
- 每个用户对同一学生只能评分一次

### 搜索和筛选
- 支持按学校、年级、班级、姓名搜索
- 支持按平均分区间筛选
- 支持按平均分、评分数量、姓名排序

### 排行榜
- 综合榜：显示 Top 10 学生
- 学校榜：显示 Top 10 学校
- 支持按周期筛选（全部、本周、本月、本季度）

## 注意事项

- 本项目为纯前端实现，所有数据存储在内存中（使用 Pinia store）
- 刷新页面后数据会重置（除了登录状态会保存在 localStorage）
- 实际项目中需要连接后端 API 进行数据持久化

## 评分等级说明

- **夯** (5分)：最高评价
- **顶级** (4分)：优秀
- **人上人** (3分)：良好
- **NPC** (2分)：一般
- **拉完了** (1分)：最低评价
